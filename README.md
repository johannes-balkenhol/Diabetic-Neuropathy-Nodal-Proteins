# Diabetic Neuropathy — Nodal Protein Periodicity Analysis

Analysis code for a study quantifying paranodal protein periodicity
in diabetic neuropathy using dSTORM autocorrelation maps.

**Status:** Paper under review — results not disclosed pre-publication.  
**Data:** Raw imaging data will be deposited in BioImage Archive upon publication.  
**Code:** Available here for review and reproducibility.

---

## What this code does

Automated pipeline to:
1. Load dSTORM autocorrelation TIFFs (Caspr1, PanNF markers)
2. Extract periodicity measurements from correlation maps
3. Compute midline profile features (kurtosis, spectral entropy, etc.)
4. Correlate structural measures with clinical scores

## Repository structure
```
code/
  acf_pipeline/          Main pipeline: TIFF → ACF scores
  feature_analysis/      Midline profile feature extraction
  clinical_correlation/  Spearman correlations + publication figures
  utils/                 Shared helpers

  methods.docx           Detailed methods
envs/
  acf_env.yml            Reproducible conda environment
results/
  figures/publication/   Final figures (generated by pipeline)
```

## Installation
```bash
git clone git@github.com:johannes-balkenhol/Diabetic-Neuropathy-Nodal-Proteins.git
cd Diabetic-Neuropathy-Nodal-Proteins

conda env create -f envs/acf_env.yml
conda activate acf_env
```

## Usage
```bash
# Step 1 — ACF pipeline (requires raw TIFFs from BioImage Archive)
jupyter lab code/acf_pipeline/acf_v4_final.ipynb

# Step 2 — Feature analysis
jupyter lab code/feature_analysis/profile_feature_extraction.ipynb

# Step 3 — Clinical correlations
jupyter lab code/clinical_correlation/feature-ACF-correlation-clean.ipynb
```
